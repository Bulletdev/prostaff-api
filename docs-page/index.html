ProStaff API
Esports Team Management Platform — REST API Reference v1

What is ProStaff?

ProStaff is a full-stack esports management platform for League of Legends organizations. It provides player analytics, draft intelligence, scrim management, inhouse leagues, competitive intelligence, contract management, and real-time team communication.

API Overview

Rails 7.1 REST API with JWT authentication, multi-tenant architecture (organization-scoped data), WebSocket channels via Action Cable, and background job processing via Sidekiq.

Tech Stack

Ruby 3.4 · Rails 7.1 · PostgreSQL 15 · Redis 7 · Sidekiq · Meilisearch · Action Cable · Supabase

External APIs

Riot Games API · PandaScore · ProStaff Scraper · Leaguepedia · OpenAI · Discord · AWS S3 · ProPay (PIX)

All endpoints require Authorization: Bearer {access_token} unless marked as public. Multi-tenant: every query is automatically scoped to the authenticated organization.
Subscription Tiers
Tier 3 — Amateur Tier 2 — Semi-Pro Tier 1 — Professional
Authentication
JWT Bearer tokens — access (24h) + refresh (7 days)

ProStaff uses JWT tokens. Login returns an access_token (24h) and a refresh_token (7 days). Send the access token in every request header. When it expires, use the refresh endpoint to get a new pair without re-authenticating.

    Every Request
    
# HTTP header (all protected endpoints)
Authorization: Bearer {access_token}

# WebSocket connection
wss://api.prostaff.gg/cable?token={access_token}

  


  

    JWT payload includes: user_id, organization_id, role, jti (for blacklisting), tier.
    Player tokens additionally carry entity_type: "player" and player_id.

 Arquitetura — 8 Servicos

  
Stack completa do ecossistema ProStaff

  

    

      
prostaff-api

      :3333
      
Ruby 3.4 · Rails 7.2.3.1 · PostgreSQL 15 · Redis 7 · Sidekiq

      
Core API — auth JWT, multi-tenant (organization_scoped), Action Cable WebSockets, Meilisearch, Pundit, Blueprinter. Deploy via Coolify (SA East).

    

    

      
ProPay

      :5555
      
Ruby · Roda 3.103 · PostgreSQL · Redis

      
Servico de carteira digital. PIX deposits via Abacate Pay, gestao de saldo, payouts. API REST isolada, consumida via proxy no prostaff-api (/api/v1/wallet).

    

    

      
prostaff-riot-gateway

      :4444
      
Go 1.23 · HTTP proxy · Internal JWT

      
Proxy seguro para a Riot Games API. Autentica requisicoes internas via INTERNAL_JWT_SECRET compartilhado com o prostaff-api. Rate limiting e region routing.

    

    

      
prostaff-events

      :4000
      
Elixir · Phoenix 1.7 · Phoenix Channels · Redis pub/sub

      
Bus de eventos de dominio em tempo real. EventPublisher via Redis, InhouseQueue GenServer, DraftChannel. 10 domain events wired (player_synced, match_imported, contract_expiring…).

    

    

      
prostaff-ml

      :8001
      
Python · FastAPI · scikit-learn · pandas

      
Servico de Machine Learning. Modelos de win probability, analise de draft avancada, feature engineering sobre dados de partidas. Alimenta o modulo ai_intelligence da API Rails.

    

    

      
ProStaff-Scraper

      :8000
      
Python · FastAPI · Elasticsearch 8.13 · Leaguepedia

      
Pipeline ETL de partidas profissionais. Scraping de LoL Esports API + Leaguepedia + Riot Match-V5. 97K+ jogos pro indexados no Elasticsearch. Backfill historico e enrichment daemon.

    

    

      
prostaff-discord-bot

      Railway
      
Ruby · discordrb 3.5 · SQLite · Scheduler

      
Bot Discord com slash commands. Inhouse queue server-side, scrims via Discord, matchmaking, webhooks, embeds. Integra com a prostaff-api via REST. Deploy no Railway.

    

    

      
prostaff-analytics-hub

      :3000
      
Next.js 15 · React 19 · TanStack Query · Zustand · Framer Motion

      
Frontend principal. Dashboard, analytics, scouting, draft board, tactical board (React-Konva). Internacionalizacao em 5 idiomas (next-intl). Auth via cookies JWT, shadcn/ui + Tailwind.
